# 许可证正文说明

本目录保存项目使用的标准许可证正文。

## 为什么保留英文官方文本

Apache-2.0 和 MPL-2.0 的法律解释通常以官方英文文本为准。为了减少误解，本仓库保留官方英文许可证正文，并在仓库根目录用中文文件说明项目边界。

CC BY-SA 4.0 有官方简体中文法律文本，可在 Creative Commons 官网查看：

- https://creativecommons.org/licenses/by-sa/4.0/legalcode.zh-hans

## 本项目使用的许可证

- [Apache-2.0.txt](Apache-2.0.txt)：用于大部分软件、SDK、ROS 2、Web、示例、工具和 URDF 运动学资源。
- [MPL-2.0.txt](MPL-2.0.txt)：用于 `firmware/` 下项目原创固件源码。
- [CC-BY-SA-4.0.txt](CC-BY-SA-4.0.txt)：用于文档和脱敏后的公开 mesh 资产。
