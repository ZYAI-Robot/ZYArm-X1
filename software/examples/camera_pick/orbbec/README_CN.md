# 机器人手臂抓取示例

## 概述

本示例演示了如何使用奥比中光SDK实现机器人抓取应用。该应用显示相机的彩色图像，允许用户在图像上点击选择目标点，计算点击点的3D坐标，并将这些坐标发送给机器人手臂执行抓取动作。

## 功能

- 显示奥比中光相机的实时彩色图像
- 支持在彩色图像上通过鼠标点击选择点位
- 使用深度信息计算点击像素的3D世界坐标
- 通过串口通信控制机器人手臂执行抓取动作

## 实现细节

1. 应用程序创建一个管道从相机捕获彩色流和深度流
2. 启用从深度到彩色的软件对齐以确保坐标一致性
3. 彩色图像显示在OpenCV窗口中
4. 当用户点击图像时，应用程序：
   - 获取点击的像素坐标
   - 检索该像素对应的深度值
   - 将2D像素坐标加深度转换为3D世界坐标
   - 将3D坐标转换为机器人坐标系
   - 通过串口发送控制命令给机器人手臂执行抓取动作

## 使用方法

1. 确保机器人通过USB转串口设备连接到计算机（默认使用/dev/ttyUSB0）
2. 构建并运行应用程序
3. 当彩色图像窗口出现时，点击图像中可见的任何物体
4. 控制台将显示点击点的3D坐标
5. 机器人手臂将执行预定义的抓取动作
6. 按ESC键退出应用程序

## 自定义

要与实际的机器人手臂集成：

1. 配置 `zyarm_sdk::ZyArm` 的串口和固件协议参数
2. 根据实际安装位置调整坐标变换参数：
   - ry_camera: 相机绕Y轴旋转角度（弧度）
   - h_camera_base: 相机到基座的垂直高度（mm）
   - l_base_robot: 基座到机械臂底座的水平距离（mm）
3. 根据需要修改robotArmPick函数中的机器人动作序列

## 坐标系统

### 相机坐标系
3D坐标在相机坐标系中提供：
- X：水平轴（向右为正）
- Y：垂直轴（向下为正）
- Z：深度轴（向前为正）

### 机器人坐标系转换
代码中通过transformPointCloudToRobotBase函数实现坐标系转换：
1. 首先将点云坐标系转换为相机坐标系
2. 然后应用旋转变换（绕Y轴）
3. 最后应用平移变换

单位为毫米。
