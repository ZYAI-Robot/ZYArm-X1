# 快速上手

本章面向第一次接触 ZYArmV1 / `ZYArm-X1` 的用户，推荐先使用官方在线 Web 控制台完成第一次连接、状态确认、复位和小幅真机动作。

官方 Web 地址：

```text
https://arm.zyairobot.com/#/zy
```

这条路线不要求你先安装 Python SDK、ROS 2、MoveIt、Gazebo 或 LeRobot，也不要求一开始手动输入串口命令。你只需要准备好机械臂、电源、USB 数据线和支持 Web Serial 的浏览器。

需要特别记住：Web 控制台不是绕过串口和固件。它是在浏览器里通过 Web Serial 连接本地机械臂串口，把页面操作转换成固件命令。因此机械臂仍然需要上电、被电脑识别、串口未被其他软件占用，并且真实动作前仍然要确认安全空间。

## 先认识这些词

| 术语 | 本章里的含义 |
| --- | --- |
| 官方 Web 控制台 | 项目方部署的在线页面，地址是 `https://arm.zyairobot.com/#/zy` |
| Web Serial | 浏览器访问本地串口的能力，用来连接真实机械臂 |
| 串口 | 电脑和控制板之间的文本通信通道，Web 和串口工具都会用到它 |
| CH340 | 控制板常用的 USB 转串口芯片，Windows 可能需要驱动 |
| `COMx` / `/dev/ttyUSBx` | Windows / Ubuntu 看到的串口设备名 |
| 模型预览 | 只更新网页里的 3D 模型，不会驱动真实机械臂 |
| 真机发送 | 页面通过串口向固件发送命令，真实机械臂会运动 |
| `230400, 8N1` | 串口工具备用路线使用的通信参数 |
| `CMD` / `STATUS` | 固件命令和状态返回，排障时会看到 |

如果某个词看不懂，可以先查 [术语与工具速查](../术语与工具速查.md)。

## 本章主线

```text
准备机械臂、电源、USB 数据线和安全空间
  -> 使用 Chrome / Edge 打开 https://arm.zyairobot.com/#/zy
  -> 给机械臂上电并连接 USB
  -> 在官方 Web 控制台中选择本地机械臂串口
  -> 确认页面显示设备已连接
  -> 确认状态同步正常
  -> 先调整草稿，只观察模型预览
  -> 确认安全后，发送一个小幅真机动作
  -> 练习停止、复位和直接断电的边界
  -> 完成第一次上手
```

## 串口工具什么时候用

串口工具不是第一选择，但它是很重要的备用路线和排障路线。遇到下面情况时，可以切换到串口工具：

- 官方 Web 无法访问。
- 浏览器不是 Chrome / Edge，或不支持 Web Serial。
- 点击“请选择设备”后看不到机械臂串口。
- 串口被其他软件占用，需要确认端口是否真的可用。
- Web 页面连接失败、状态异常，想直接查看固件 ACK 和 `[STATUS]`。
- 你想学习底层协议，理解页面操作背后的 `CMD` 指令。

串口备用路线的最小闭环是：

```text
打开串口：230400, 8N1, no flow control
  -> 发送 CMD6 查询状态：[CMD][6]
  -> 发送 CMD1 复位到初始姿态：[CMD][1]
  -> 发送 CMD0 IK 小动作：[CMD][0][200 0 0 0 0 0]
  -> 再次发送 CMD6，记录 J0-J5 和 CLAW
  -> 发送 CMD1 复位
  -> 发送 CMD3，把刚刚记录的状态发回去
  -> 发送 CMD1 再次复位
  -> 异常时优先直接断电；串口已连通且来得及时可发送 CMD2
```

## 你将得到什么

完成本章后，你应该可以：

- 使用官方 Web 控制台打开机械臂页面。
- 知道 Web 连接真实机械臂时仍然依赖本地 USB 串口和固件。
- 在浏览器里选择机械臂串口，并确认设备已连接。
- 区分“只影响模型预览”和“会驱动真机”的操作。
- 发送一次低风险、小幅度的真机动作。
- 使用停止、复位和直接断电处理异常。
- 知道何时切换到串口工具，并理解 `CMD6`、`CMD1`、`CMD0`、`CMD3`、`CMD2` 的最小排障闭环。

## 安全提醒

第一次动作前请确认：

- 机械臂周围没有人手、线材、杯子或其他障碍物。
- 你知道如何快速切断机械臂电源。
- 页面已经显示设备已连接，或串口工具已经成功读到 `[STATUS]`。
- 你明确知道当前按钮是模型预览，还是会发送真实动作。
- 如果机械臂姿态明显异常，先不要发送动作命令。

动作过程中如果出现异常，优先点击页面里的“停止”；如果停止无效或来不及操作，直接切断机械臂电源。串口已经连通且来得及时，也可以发送 `[CMD][2]` 指令急停。
