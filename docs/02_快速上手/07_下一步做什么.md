# 下一步做什么

如果你已经完成状态读取、IK 小动作、状态记录、复位和 `CMD3` 状态复现，说明真机串口链路和固件基础控制已经跑通。

## 你已经验证了什么

- 串口软件可以连接机械臂。
- `CMD6` 可以读取 `[STATUS]`。
- `CMD1` 可以让机械臂回到初始姿态。
- `CMD0` 可以通过固件 IK 控制机械臂运动。
- `CMD3` 可以复现记录到的关节状态。
- `CMD2` 可以作为动作异常时的指令急停。
