# 安装完成检查

本页用于在进入具体玩法或框架前做最后确认。不同目标只需要检查对应部分。

如果某个检查项里的工具名不熟悉，先回到对应环境页面，或查 [术语与工具速查](../术语与工具速查.md)。本页只做最终确认，不替代安装教程。

## 官方在线 Web 快速使用

这部分面向第一次使用机械臂的学习者，用来确认官方 Web 页面和真实机械臂连接条件都已经准备好。

- [ ] 可以访问 `https://arm.zyairobot.com/#/zy`。
- [ ] 正在使用 Chrome 或 Edge。
- [ ] 机械臂已经正确上电。
- [ ] 已知道如何快速切断机械臂电源。
- [ ] USB 数据线支持数据传输。
- [ ] Windows 能看到 `COMx`，或 Ubuntu 能看到 `/dev/ttyUSBx` / `/dev/ttyACMx`。
- [ ] 串口没有被串口助手、Keil、Python 脚本或其他浏览器页面占用。
- [ ] Web 页面可以选择设备并显示“设备已连接”。
- [ ] 页面状态没有持续异常。
- [ ] 已完成一次模型草稿、小幅真机发送、停止和复位练习。

下一步：[快速上手](../02_快速上手/README.md) 或 [Web 控制台入门](../05_常用玩法/01_Web仿真体验.md)。

## 串口工具备用检查

- [ ] 机械臂已经正确上电。
- [ ] 已知道如何快速切断机械臂电源。
- [ ] Windows 能看到 `COMx`，或 Ubuntu 能看到 `/dev/ttyUSBx` / `/dev/ttyACMx`。
- [ ] 串口参数为 `230400, 8N1, no flow control`。
- [ ] 可以发送 `[CMD][6]` 并看到 `[STATUS]`。
- [ ] 可以发送 `[CMD][1]` 让机械臂复位。

下一步：[快速上手](../02_快速上手/README.md) 或 [基础操作](../04_基础操作/README.md)。

## 官方 Web 控制台补充检查

这部分用于课程、展会或比赛现场。目标是确认官网在线页面和本地串口链路都可用。

- [ ] 现场网络可以访问官方地址。
- [ ] 浏览器可以正常显示 3D 模型和功能选项卡。
- [ ] 浏览器能弹出 Web Serial 串口选择窗口。
- [ ] 页面连接设备后能读取状态。
- [ ] 已准备串口工具作为官网不可用时的备用路线。
- [ ] 已记录问题反馈所需信息：浏览器版本、系统、串口名、页面提示和固件版本。

下一步：[Web 控制台使用环境](02_Web仿真环境.md)、[Web 控制台入门](../05_常用玩法/01_Web仿真体验.md) 或 [Web 接入](../06_仿真与框架接入/Web/README.md)。

## 遇到问题时

如果复杂框架无法运行，先回退检查：

```text
机械臂上电
  -> USB 串口可见
  -> 官方 Web 可连接，或串口工具 CMD6 可读状态
  -> CMD1 或页面复位可用
  -> Python / 官方 Web / ROS / LeRobot 环境验证
  -> 目标玩法或实验
```

排障入口：[校准维护与排障](../08_校准维护与排障/README.md)。
