# 常见扩展场景

本页面向需要修改 `lerobot_robot_zyarm` 插件的开发者。完整采集命令和使用流程见 [科研与数据采集](../../07_科研与数据采集/README.md)，这里关注“改源码时要动哪些文件、检查哪些风险”。

## 风险分层

| 风险 | 场景 | 主要检查 |
| --- | --- | --- |
| 低 | 补文档、补测试、补注释 | 不改变字段和运行行为 |
| 中 | 增加 camera 配置说明、增加 observation 字段 | dataset、训练读取、测试 |
| 高 | 修改 action 字段、单位、夹爪范围、滤波、LeRobot API | replay、policy、兼容性、真实机械臂安全 |

## 增加 observation 字段

可能涉及：

- `features.py`：声明新字段。
- `robot.py`：在 `get_observation()` 中填充字段。
- `tests/test_conversion.py` 或 `tests/test_robot.py`：增加字段断言。
- `docs/07_科研与数据采集`：如果用户会看到该字段，需要补说明。

注意：observation 字段会进入 dataset。字段名、类型和 shape 改动会影响后续训练。

## 修改 action 表达

可能涉及：

- `features.py`：修改 `ACTION_KEYS`。
- `conversion.py`：修改 `action_to_positions()` 和 `positions_to_action()`。
- `robot.py`：确认 `send_action()` 下发仍然安全。
- `teleoperator.py`：确认 `get_action()` 输出字段一致。
- tests：至少检查 conversion、robot、teleoperator。

高风险点：

- policy 输出维度可能变化。
- 旧 dataset 的 replay 可能失效。
- 夹爪范围变化可能导致真实夹爪动作异常。

## 增加或改变相机配置

当前 camera feature 由 `camera_features()` 根据 config 的 `height` 和 `width` 生成：

```python
features[str(name)] = (int(height), int(width), 3)
```

开发时需要检查：

- camera 名称是否稳定，例如 `front`、`wrist`。
- shape 是否和实际图像一致。
- `robot.py` 中 `camera.read_latest()` 是否返回 LeRobot 期望的图像格式。
- 使用教程是否需要更新 [摄像头与视角配置](../../07_科研与数据采集/04_摄像头与视角配置.md)。

## 调整滤波或频率参数

相关字段：

- follower: `state_max_age_ms`、`slave_filter_lpf_alpha`
- leader: `leader_hz`、`action_max_age_ms`、`wait_timeout_ms`

相关文件：

- `config.py`
- `robot.py`
- `teleoperator.py`
- `tests/test_config.py`
- `tests/test_robot.py`
- `tests/test_teleoperator.py`

调整这些参数会影响手感、延迟、数据质量和异常恢复。不要只看命令能不能启动，也要做短时间 teleoperate、record、replay。

## 修改安全行为

`robot.py` 的 action 下发前会经过：

```python
safe_positions = self.safety.sanitize_positions(positions)
```

如果修改 `SafetyConfig` 或 `SafetyController` 相关行为，需要同步检查：

- SDK 安全配置。
- follower 是否仍能防止异常 action。
- replay 和 policy eval 是否会产生危险动作。
- [测试发布与贡献检查](../09_测试发布与贡献检查.md) 中的真实设备验证记录。

## 升级 LeRobot

升级 LeRobot 时，至少检查：

- `RobotConfig.register_subclass` 和 `TeleoperatorConfig.register_subclass` 是否仍可用。
- `Robot` / `Teleoperator` 抽象类接口是否变化。
- camera config 和 `make_cameras_from_configs()` 是否变化。
- `RobotAction` / `RobotObservation` 字段期望是否变化。
- `lerobot-teleoperate`、`lerobot-record`、`lerobot-replay` 命令参数是否变化。
- dataset 写入和 replay 读取行为是否变化。
