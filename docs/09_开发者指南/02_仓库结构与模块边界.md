# 仓库结构与模块边界

本页帮助维护者定位“这个能力应该在哪个目录里改”。代码目录 README 讲源码细节，本页只讲边界和影响范围。

## 顶层目录

| 目录 | 负责范围 | 不建议放入 |
| --- | --- | --- |
| `firmware/` | STM32 固件、串口命令、运动控制、状态上报 | 上位机 SDK 示例、课程文档 |
| `hardware/` | 硬件参考资料、模型和配置资料 | 运行日志、临时数据 |
| `software/` | SDK、ROS 2、LeRobot、示例、工具、诊断 | 固件工程文件 |
| `docs/` | 面向使用者和开发者的文档 | 构建产物和临时实验数据 |
| `data/` | 本地学习数据、运行日志 | 需要作为源码维护的程序 |
| `openspec/` | 需求、设计、任务和变更记录 | 运行时代码 |

## software 内部边界

| 目录 | 主要职责 | 变更后通常要检查 |
| --- | --- | --- |
| `software/zyarm_sdk/` | Python/C++ SDK、协议 contracts、API 测试 | 基础操作、Serial API、tools、diagnostics、LeRobot |
| `software/ros2_ws/` | ROS 2 工作区、MoveIt、Gazebo、ros2_control | 安装文档、仿真接入、真机 launch |
| `software/lerobot_robot_zyarm/` | LeRobot 原生插件 | 科研与数据采集、外设连接、SDK 单位 |
| `software/examples/` | 面向学习和应用的示例程序 | 示例 README、依赖说明、运行截图 |
| `software/tools/` | 日常调试、信息读取、遥操工具 | 安装文档、常用玩法、排障文档 |
| `software/diagnostics/` | 压测、可靠性验证、故障定位 | 排障文档、发布检查 |

## 固件边界

固件决定真实机械臂会怎样动。常见入口包括：

- `firmware/Core/Inc/arm_shell.h`：`CMD_ID` 枚举、命令包结构、解析格式。
- `firmware/Core/Src/arm_shell.c`：命令解析、队列、执行器、ACK/响应输出。
- `firmware/Core/Src/arm_shell_cmd.c`：各个 `[CMD]` 的 handler 和帮助信息。
- `firmware/Core/Src/arm_robot.c`：机械臂基础动作和状态。
- `firmware/Core/Src/arm_robot_kinematics.c`：运动学相关逻辑。
- `firmware/Core/Src/arm_robot_recorder.c`：动作录制与回放。
- `firmware/Core/Src/arm_remote.c`：遥控模式。
- `firmware/Core/Src/arm_master_slave.c`：主从跟随模式。
- `firmware/Core/Src/arm_status_report.c`：自动状态上报。
- `firmware/Core/Src/arm_flash.c`：Flash 存储相关逻辑。

## 生成物和临时文件

- `software/ros2_ws/build/`、`install/`、`log/` 是 `colcon` 生成物。
- SDK 的临时 build 目录不应作为源码提交。
- 本地数据和日志默认放在 `data/learning/`、`data/logs/`。

## 边界判断示例

| 需求 | 首要修改位置 | 可能联动 |
| --- | --- | --- |
| 新增一个固件命令 | `firmware/Core/Src/arm_shell_cmd.c` | 基础操作、Serial API、SDK、tools、diagnostics |
| 增加 Python API | `software/zyarm_sdk/python/` | SDK tests、示例、LeRobot |
| 增加 MoveIt 真机启动参数 | `software/ros2_ws/src/zyarm_moveit_config/` | ROS 2 文档、安装检查 |
| 改变会影响官方 Web 的协议字段 | `firmware/`、`software/zyarm_sdk/` 或相关上层模块 | Web 接入、Web 控制台入门、截图和排障文档 |
| 增加数据采集配置 | `software/lerobot_robot_zyarm/` | 科研与数据采集、外设连接 |
