# 复位停止与安全动作

本页整理复位、停止、零点和卸力相关指令。它们直接影响机械臂姿态和输出状态，操作前请确保机械臂周围没有障碍物。

## 优先安全原则

- 异常运动、碰撞风险或线材被拉扯时，优先直接切断机械臂电源。
- 串口已经连通且来得及发送时，可以发送 `[CMD][2]` 指令急停。
- 不确定当前姿态是否安全时，先不要发送动作指令。
- `zero` 和 `poweroff` 不属于普通入门动作，使用前应理解后果。

## CMD1 reset

用途：让机械臂回到固件定义的初始姿态。

```text
[CMD][1]
```

典型流程：

```text
[CMD][6]
[CMD][1]
[CMD][6]
```

使用注意：

- 快速上手和多数实验开始前，都建议先执行一次 `CMD1`。
- 复位过程中保持机械臂运动空间清空。
- 如果复位方向明显异常，立即断电并排查装配、零点或固件版本。

## CMD2 stop

用途：指令急停，立即停止当前运动并锁定关节。

```text
[CMD][2]
```

使用注意：

- `CMD2` 在固件解析任务中直接处理，可能不会返回普通指令那样的完整 ACK。
- 它适合串口仍然连通、且你能快速发送指令的场景。
- 对初学者而言，最可靠、最直观的安全动作仍然是直接切断机械臂电源。

## CMD7 zero

用途：将当前位置设为零点。

```text
[CMD][7]
```

使用注意：

- 这是维护和校准类指令，不建议在普通实验中随意使用。
- 错误零点会影响后续复位、角度控制和 IK 控制。
- 使用前应确认机械臂已经处于项目方推荐的零点姿态。

> 待项目方补充：请提供标准零点姿态照片、零点校准流程和误设置后的恢复方法。

## CMD23 poweroff

用途：让关节停止输出或卸力。固件命令名称为 `poweroff`。

```text
[CMD][23]
```

使用注意：

- 该指令不是切断外部电源，而是通过固件让关节进入停止输出或卸力状态。
- 机械臂可能因重力或外力移动，使用前请扶稳机械臂或确认姿态安全。
- 真正需要断电时，请关闭机械臂供电。
