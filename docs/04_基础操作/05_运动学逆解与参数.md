# 运动学逆解与参数

本页整理运动学逆解、IK 权重和遥控阈值相关指令。快速上手中的第一次低风险动作使用的就是 `CMD0`。

IK 是 inverse kinematics 的缩写，也就是运动学逆解。你给出末端目标位姿，固件尝试计算 6 个关节应该转到什么角度。

## CMD0 ik_inverse

用途：根据末端目标位姿计算关节角度，并由固件控制机械臂运动。

```text
[CMD][0][x y z rx ry rz]
```

可选追加 `verbose`：

```text
[CMD][0][200 0 0 0 0 0 1]
```

参数说明：

| 参数 | 含义 |
| --- | --- |
| `x y z` | 末端目标位置 |
| `rx ry rz` | 末端目标姿态 |
| `verbose` | 可选，非 0 时输出 IK 计算得到的关节结果 |

`x y z` 描述末端在空间中的目标位置，`rx ry rz` 描述末端姿态。本页暂不展开完整坐标系，首次使用只建议复制文档中的低风险示例。

入门示例：

```text
[CMD][1]
[CMD][0][200 0 0 0 0 0]
[CMD][6]
```

使用注意：

- 这是比手动填写 6 个关节角更容易上手的动作方式。
- 目标位姿超出机械臂工作空间时会执行失败。
- 首次使用只发送文档中确认过的小动作，不要直接尝试大范围位姿。

## CMD27 get_weights

用途：读取 IK 关节权重。

```text
[CMD][27]
```

典型返回包含：

```text
Joint Weights: [...]
```

使用注意：

- 权重会影响 IK 求解偏好。
- 普通学习者只需要读取，不建议修改。

IK 权重可以理解成“求解时更偏向或更不偏向使用某些关节”。它属于调参项，不是常规控制命令。

## CMD28 set_weights

用途：设置 IK 关节权重。

```text
[CMD][28][w0 w1 w2 w3 w4 w5]
```

参数说明：

| 参数 | 含义 |
| --- | --- |
| `w0..w5` | 6 个关节的 IK 权重 |

使用注意：

- 这是算法和调参类指令，可能改变 IK 解的关节分布。
- 不建议在课程入门或首次上手阶段使用。
- 修改前先记录 `[CMD][27]` 的原始输出。

> 待项目方补充：请提供默认权重、推荐调参范围和典型任务下的权重配置建议。

## CMD29 set_angle_threshold

用途：设置遥控模式下的角度阈值。

```text
[CMD][29][threshold]
```

使用注意：

- 该阈值主要服务遥控链路。
- 默认值来自固件当前实现，普通用户不需要修改。
- 修改后可用 `CMD30` 确认。

阈值可以理解成过滤或判断动作变化的门槛。门槛太小可能更敏感，门槛太大可能响应变慢，具体推荐值需要结合遥控工具验证。

## CMD30 get_angle_threshold

用途：读取遥控模式下的角度阈值。

```text
[CMD][30]
```

典型返回：

```text
ACK_RESPONSE: CMD_ID=30, ANGLE_THRESHOLD:2.00
```

使用注意：

- 如果遥控动作过滤或响应异常，可以记录该值用于排障。
- 具体阈值含义和推荐范围需要结合遥控章节和项目方参数说明。
