# 项目定位与适用场景

ZYArmV1 是一套面向教育、创客、比赛和科研的 6+1 轴桌面机械臂平台。它强调真实硬件、低成本、代码开放和多层软件接入，适合从串口控制一路学习到 Web、ROS 2、MoveIt、Gazebo、LeRobot 和数据采集。

如果你第一次看到这些名称，可以先这样理解：固件和串口协议负责“让机械臂听懂命令”，SDK 让 Python/C++ 程序更容易控制机械臂，Web 用于浏览器可视化和交互，ROS 2 / MoveIt / Gazebo 面向机器人系统、规划和仿真，LeRobot 面向遥操、数据采集、回放和策略评估。更完整解释见 [术语与工具速查](../术语与工具速查.md)。

## 它是什么

- 一套真实可运行的 6+1 轴桌面机械臂。
- 一套从固件、串口协议、SDK 到 Web、ROS 2、LeRobot 都尽量开放的教学工程。
- 一条从最小串口命令开始，再逐步进入仿真、遥操、数据采集和算法实验的学习路径。

## 它不是什么

- 不是只用于展示的封闭玩具机械臂。
- 不是只提供仿真、不连接真实硬件的算法项目。
- 不是要求用户一开始就安装 ROS 2、MoveIt、Gazebo 或 LeRobot 的复杂科研平台。

## 核心优势

- 标准 6+1 轴：六个机械臂关节加一个夹爪自由度，适合讲解关节控制、末端执行器控制、运动学逆解、轨迹执行和遥操。
- 低成本真实硬件：让学习者和创客先接触能真实运动、能真实排障的机械臂，而不是只停留在仿真环境。
- 代码开放：仓库覆盖 `firmware/`、`software/zyarm_sdk/`、`software/ros2_ws/`、`software/lerobot_robot_zyarm/`、`web/`、示例工具和诊断脚本，便于按目标切入二次开发。
- 路线可渐进：第一次上手只需要跑通串口状态读取和小幅动作；后续再进入 SDK、Web、ROS 2、MoveIt、Gazebo、LeRobot 或算法实验。
- 适合具身智能实验：SDK 和 LeRobot 路径支持遥操、数据采集、回放、策略评估，以及模仿学习、强化学习、VLA 等方向的前期验证。

## 推荐起点

无论最终目标是什么，都建议先完成一次最小真机闭环：

```text
快速上手
  -> 基础操作
  -> 常用玩法与项目案例 / 仿真与框架接入 / 科研与数据采集
```

这样可以先确认硬件、固件、串口和上位机链路是通的，再进入 Web、ROS 2、LeRobot 或算法实验，排障会简单很多。
