# 适用人群

不同用户可以从不同入口开始，不需要把所有框架一次学完。

| 人群 | 典型目标 | 建议先看 | 暂时不用先看 |
| --- | --- | --- | --- |
| 学习者 | 第一次让真实机械臂动起来，理解串口、关节、夹爪和状态读取 | [快速上手](../02_快速上手/README.md)、[基础操作](../04_基础操作/README.md) | ROS 2、MoveIt、Gazebo、LeRobot |
| 创客和兴趣爱好者 | 做桌面抓取、遥操、动作录制、Web 仿真和自动化小项目 | [常用玩法与项目案例](../05_常用玩法/README.md)、[仿真与框架接入](../06_仿真与框架接入/README.md) | 固件内部实现 |
| 比赛用户 | 快速搭建抓取、分拣、遥操和视觉任务原型 | [常用玩法与项目案例](../05_常用玩法/README.md)、[校准维护与排障](../08_校准维护与排障/README.md) | 深入 SDK 架构 |
| 教师和课程维护者 | 组织串口控制、运动学、仿真、数据采集和模仿学习实验 | [产品简介](README.md)、[后续文档计划](../后续文档计划.md) | 单个源码文件细节 |
| 科研人员 | 做主从遥操、LeRobot 数据采集、策略回放、模仿学习、强化学习和 VLA 实验 | [科研与数据采集](../07_科研与数据采集/README.md) | Web 前端实现细节 |
| 二次开发者 | 修改固件协议、SDK、ROS 2 控制链路、Web 控制端或新增工具 | [开发者指南](../09_开发者指南/README.md)、代码目录 README | 面向首次上手的解释性内容 |

## 共同的推荐起点

无论最终目标是什么，都建议先完成一次最小真机闭环：

```text
快速上手
  -> 基础操作
  -> 常用玩法与项目案例或框架接入
```

这样可以先确认硬件、固件、串口和上位机链路是通的，再进入 Web、ROS 2、LeRobot 或算法实验。
