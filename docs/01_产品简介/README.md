## 先认识这些词

| 术语                      | 你可以先这样理解                          |
| ----------------------- | --------------------------------- |
| 固件                      | 跑在控制板上的底层程序，负责接收串口命令并控制关节         |
| SDK                     | 给 Python/C++ 程序使用的控制封装，避免每次手写串口命令 |
| Web                     | 浏览器里的控制端或仿真界面，适合观察模型和可视化交互        |
| ROS 2 / MoveIt / Gazebo | 机器人系统、运动规划和仿真工具栈，适合进阶控制与仿真        |
| LeRobot                 | 面向遥操、数据采集、回放和策略评估的机器人学习工具链        |
| 100Hz                   | 每秒 100 次控制或状态读取的高频控制目标量级          |
| VLA                     | 结合视觉、语言和动作输出的具身智能模型方向             |

更多解释见 [术语与工具速查](../术语与工具速查.md)。

## 一句话定位

ZYArmV1 是一套面向教育、创客、比赛和科研的 6+1 轴桌面机械臂平台。它强调真实硬件、低成本、代码开放和多层软件接入，适合从串口控制一路学习到 Web、ROS 2、MoveIt、Gazebo、LeRobot 和数据采集。
