# 核心优势

## 标准 6+1 轴

机械臂提供六个机械臂关节和一个夹爪自由度，适合教学中讲解关节控制、末端执行器控制、运动学逆解、轨迹执行和遥操。

为什么重要：

- 学习者可以从单关节控制逐步过渡到多关节同步控制。
- 课程可以覆盖关节空间、末端空间、夹爪控制和简单任务规划。
- 科研用户可以把六轴动作和夹爪状态记录成统一的 action/observation。

这里的 `action` 可以理解为“发给机械臂的动作指令”，`observation` 可以理解为“机械臂状态和摄像头画面等观测信息”。科研框架通常会把这两类数据一起保存，用于模仿学习或策略评估。

## 超高性价比

项目目标是让学习者和创客以较低成本接触真实机械臂，而不是只停留在仿真环境。

___2600+机械臂 VS 万元体验___

## 代码开放，适合二次开发

仓库覆盖：

- `firmware/`：STM32 固件工程
- `software/zyarm_sdk/`：Python/C++ SDK
- `software/ros2_ws/`：ROS 2、MoveIt、Gazebo 相关包
- `software/lerobot_robot_zyarm/`：LeRobot 原生适配
- `web/`：Web 控制端和仿真界面
- `software/examples/`、`software/tools/`、`software/diagnostics/`：示例、工具和压测诊断

为什么重要：

- 初学者可以先使用文档和示例，不必读源码。
- 二次开发者可以从 SDK、ROS 2、Web 或固件任一层切入。
- 教师可以把同一个硬件平台拆成不同难度的实验。

## 多种控制和教学玩法

项目方向覆盖主从臂遥操、手柄/手臂遥操、动作录制、关节控制、运动学逆解、Web 仿真、视觉抓取和数据采集等能力。第一版文档会优先保证学习者能跑通基础链路，进阶玩法后续持续展开。

对应能力：

- 底层串口命令：状态读取、关节控制、夹爪控制、速度设置、录制和回放。
- 上位机工具：主从遥操、手柄控制、诊断压测。
- 框架接入：Web、ROS 2、MoveIt、Gazebo、LeRobot。

## 面向具身智能实验

SDK 和 LeRobot 路径支持高频控制和状态读取的实验方向，适合强化学习、模仿学习、VLA、大模型控制、数据集采集和策略评估。

这些方向不要求第一次上手就掌握。可以先把它们理解成：模仿学习依赖人工演示数据，强化学习依赖奖励信号，VLA 则尝试把视觉、语言和动作模型连在一起。基础串口链路稳定后，再进入这些实验会更可靠。
