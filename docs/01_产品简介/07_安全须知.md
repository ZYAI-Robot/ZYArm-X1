# 安全须知

真实机械臂会移动、夹取和碰撞物体。第一次使用时，请把安全放在动作效果前面。

## 上电前检查

- [ ] 机械臂放在稳定桌面上。
- [ ] 机械臂周围没有人手、线材、杯子或易碰撞物体。
- [ ] 电源规格和接口方向正确。
- [ ] USB 串口线不会被机械臂运动拉扯。
- [ ] 你知道如何快速断电。

> 待项目方补充：请提供推荐电源规格、上电顺序、断电方式和急停方案。

## 首次动作前检查

- [ ] 已经成功读取 `[STATUS]`。
- [ ] 已经设置保守速度或确认示例动作幅度很小。
- [ ] 机械臂不在限位附近。
- [ ] 夹爪附近没有易碎物体。
- [ ] 本次动作可以随时中断或断电。

不要在未知零位或未知限位状态下发送大幅度动作。

## 遥操和回放前检查

- [ ] 主从臂遥操前，确认 leader/follower 串口没有接反。
- [ ] 初次遥操时先空载、低速、低幅度。
- [ ] 初次回放数据集或动作录制时，先空载测试。
- [ ] 不要在人手靠近机械臂时运行自动回放或策略评估。

## 摄像头和线材

- [ ] 摄像头线材不要进入机械臂运动范围。
- [ ] 腕部摄像头安装后，不影响夹爪或关节运动。
- [ ] 线材留有余量，避免被拉扯。
- [ ] 摄像头支架固定可靠，不会在动作中松动。
- [ ] USB 串口线、摄像头线和供电线分开走线。
- [ ] 线材不要形成会被关节扫到的线圈。
- [ ] 多设备场景下，每根线都有清楚的角色标签。
- [ ] 电源适配器放在稳定位置，不悬空拉扯。
- [ ] 电源开关或断电位置没有被设备压住。

> 待项目方补充：请提供摄像头走线示例照片、推荐固定方式、双臂布线照片和电源固定建议。

## 出现异常时

- 如果机械臂不按预期运动，立即发送 `stop` 指令急停；必要时断电。
- 如果动作卡顿、抖动或方向异常，先进入 [校准维护与排障](../08_校准维护与排障/README.md)。
- 如果串口状态无法读取，不要继续发送动作命令。

## 关于指令急停

固件支持 `stop` 指令急停，用于立即停止当前运动并锁定关节。真实操作时仍建议保留物理断电或急停开关作为最后保护手段，尤其是在串口链路异常、电脑无响应或供电异常时。

> 待项目方补充：请提供 `stop` 指令急停的推荐触发方式、示例命令、适用边界，以及物理断电或急停开关的正式使用说明。
