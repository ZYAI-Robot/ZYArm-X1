## 典型能力

### 基础控制

- 读取机械臂状态、固件版本、关节角度、夹爪位置和速度。
- 复位、停止、设置零点、设置单关节角度、同步设置六个关节和夹爪。
- 设置夹爪开合、调整关节速度、执行小幅运动学逆解动作。

### 遥操与项目玩法

- 主从臂遥操和手柄遥操。
- 动作录制、动作列表查看、动作回放和清理。
- 官方 Web 模型展示、拖动交互、摄像头抓取案例和桌面分拣任务。

### 框架与算法实验

- 通过 Serial API、Python SDK 或 C++ SDK 写自己的控制脚本。
- 通过 ROS 2、MoveIt 和 Gazebo 做模型显示、规划、仿真和真机控制。
- 通过 LeRobot 做遥操数据采集、replay 回放、数据质量检查和策略评估。
