# ZYArmV1 文档中心

这里是 ZYArmV1 的文档入口，第一目标是帮助学习者、创客和客户快速把真实机械臂跑起来；第二目标是支撑科研用户进行遥操、数据采集、仿真和算法实验。

> 命名说明：当前文档面向 `ZYArm-X1` 编写；ROS 2 模型资源统一使用 `x1_standard` 命名。

## 第一次使用

如果你第一次接触这台机械臂，建议按下面顺序阅读：

```text
01_产品简介
  -> 02_快速上手
  -> 03_安装与准备
  -> 04_基础操作
  -> 08_校准维护与排障
```

这条路线不要求你先安装 ROS 2、MoveIt、Gazebo 或 LeRobot。它的目标很简单：上电、连接串口、读到状态、安全地动一下。

如果阅读过程中遇到不熟悉的工具或术语，例如 `CMD`、`STATUS`、SDK、ROS 2、MoveIt、Gazebo、LeRobot、`colcon` 或 `ffplay`，可以先看 [术语与工具速查](术语与工具速查.md)。
