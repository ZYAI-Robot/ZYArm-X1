# 术语与工具速查

本页用于快速解释文档中反复出现的工具、协议、参数和机器人术语。它不是完整教程；如果你只是想知道“这个词在 ZYArm 文档里是什么意思、我现在要用它做什么”，先看这里就够了。

## 最小上手链路

| 术语 | 简短说明 | 常见入口 |
| --- | --- | --- |
| 串口 | 电脑和控制板之间的文本通信通道，机械臂固件通过它接收命令并返回状态 | [快速上手](02_快速上手/README.md) |
| CH340 | 常见 USB 转串口芯片，Windows 可能需要安装驱动才能看到串口 | [连接电脑并找到串口](02_快速上手/03_连接电脑并找到串口.md) |
| `COMx` | Windows 下的串口名称，例如 `COM3`、`COM5` | [快速上手](02_快速上手/README.md) |
| `/dev/ttyUSBx` | Ubuntu/Linux 下常见的 USB 串口设备名，例如 `/dev/ttyUSB0` | [快速上手](02_快速上手/README.md) |
| 波特率 | 串口通信速度。本项目快速上手使用 `230400` | [连接电脑并找到串口](02_快速上手/03_连接电脑并找到串口.md) |
| `8N1` | 串口参数简写，表示 8 个数据位、无校验、1 个停止位 | [连接电脑并找到串口](02_快速上手/03_连接电脑并找到串口.md) |
| 流控 | 串口硬件或软件控制发送节奏的机制。快速上手使用 `no flow control` | [连接电脑并找到串口](02_快速上手/03_连接电脑并找到串口.md) |
| 换行 / `LF` | 固件需要用换行判断一条命令结束。串口软件发送命令后要追加 `\n` | [连接电脑并找到串口](02_快速上手/03_连接电脑并找到串口.md) |

## 固件指令

| 术语 | 简短说明 | 常见入口 |
| --- | --- | --- |
| 固件 | 运行在控制板上的底层程序，负责解析串口命令、控制关节和返回状态 | [基础操作](04_基础操作/README.md) |
| `CMD` | 固件文本命令格式，例如 `[CMD][6]` 查询状态 | [串口指令格式与回包](04_基础操作/01_串口指令格式与回包.md) |
| `STATUS` | 固件返回的状态字段，包含 `J0..J5` 和 `CLAW` | [状态查询与版本信息](04_基础操作/02_状态查询与版本信息.md) |
| `ACK` | 固件对命令的确认信息，常见有收到、返回数据、执行完成三类 | [串口指令格式与回包](04_基础操作/01_串口指令格式与回包.md) |
| `J0..J5` | 六个机械臂关节的状态或目标角度编号 | [关节与夹爪控制](04_基础操作/04_关节与夹爪控制.md) |
| `CLAW` | 夹爪状态或目标位置字段 | [关节与夹爪控制](04_基础操作/04_关节与夹爪控制.md) |
| IK / 运动学逆解 | 根据末端目标位置和姿态计算关节角度的方法 | [运动学逆解与参数](04_基础操作/05_运动学逆解与参数.md) |
| 录制 / 回放 | 固件记录一段动作，再按名称播放这段动作 | [动作录制与回放](04_基础操作/06_动作录制与回放.md) |
| `fast_io` / `joint_io_fast` | 高频控制链路，用于一次指令中快速下发关节目标并读取状态 | [关节与夹爪控制](04_基础操作/04_关节与夹爪控制.md) |

| LeRobot | 面向机器人学习的数据采集、回放、训练和策略评估工具链 | [科研与数据采集](07_科研与数据采集/README.md) |
| `teleoperate` | 遥操流程，通常用于人工控制机械臂产生动作 | [科研与数据采集](07_科研与数据采集/README.md) |
| `record` | 数据采集流程，把动作、状态和相机画面保存成数据集 | [科研与数据采集](07_科研与数据采集/README.md) |
| `replay` | 回放已采集的数据或动作，用于检查数据质量和复现流程 | [科研与数据采集](07_科研与数据采集/README.md) |
| `policy eval` | 策略评估，让训练好的策略控制机械臂并观察效果 | [科研与数据采集](07_科研与数据采集/README.md) |

| policy | 策略模型，根据观测输出机械臂动作 | [科研与数据采集](07_科研与数据采集/README.md) |

| `ffplay` | Linux 下常用的视频预览命令，本项目用它快速确认摄像头画面 | [摄像头与视角配置](07_科研与数据采集/04_摄像头与视角配置.md) |
| `udev` | Linux 设备命名规则机制，可用于固定多串口或多相机设备名 | [设备角色与端口确认](07_科研与数据采集/03_设备角色与端口确认.md) |
| 100Hz | 每秒 100 次控制或状态读取，是高频控制链路的目标量级 | [科研与数据采集](07_科研与数据采集/README.md) |
| 模仿学习 | 通过人工演示数据训练策略，让机械臂学习任务动作 | [科研与数据采集](07_科研与数据采集/README.md) |
| 强化学习 | 通过奖励信号训练策略，让机械臂在交互中学习行为 | [科研与数据采集](07_科研与数据采集/README.md) |
| VLA | Vision-Language-Action，结合视觉、语言和动作输出的机器人模型方向 | [科研与数据采集](07_科研与数据采集/README.md) |
